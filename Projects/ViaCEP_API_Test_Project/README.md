# ViaCEP API Test Project

## Overview

Projeto desenvolvido para praticar testes de APIs REST utilizando o Postman, aplicando conceitos de Software Quality Assurance (QA).

Validar o comportamento da API ViaCEP por meio de cenários positivos e negativos, documentando evidências, casos de teste e resultados.
