# ViaCEP API Test Project

## Objetivo

Validar o funcionamento da API pública ViaCEP através de testes utilizando Postman.

## Escopo

- Consulta de CEP válido
- Consulta de CEP inválido
- Validação da resposta HTTP
- Validação dos campos retornados

## Ferramentas

- Postman
- Git
- GitHub
